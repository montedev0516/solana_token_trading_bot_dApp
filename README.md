**Solana Token Trading Dapp**

# Features
1. Token List
   - Token name, symbol, price, 24-h price change
   - search/filter
2. Trading Pannel
   - Selected Token's information
3. Input Fields
   - Buy/Sell amount
   - Slippage tolerance (dropdown: 0.5%, 1%, 2%)
   - Stop-loss price
   - Buy/Sell button
4. Trading Mode Toggle
   - Auto/Manual 
5. GraphQL Client
6. Form Validation

# Required Module, library

- radix-ui
- solana/web3.js
- tanstack/react-query
- tailwindcss
